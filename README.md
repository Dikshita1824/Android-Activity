# 📱 User Registration Form 

A simple Android application that allows users to fill a registration form including personal details like name, mobile number, address, gender, DOB, and hobbies. After submission, the entered data is displayed on the screen.

---

## ✨ Features

- 👤 Enter Name
- 📞 Enter Mobile Number
- 🏠 Enter Address
- ⚧ Gender Selection (Male / Female / Other)
- 📅 DOB Selection Button
- 🎯 Hobbies Selection (Checkboxes)
- 📤 Submit Button
- 🧾 Display Output in TextView
